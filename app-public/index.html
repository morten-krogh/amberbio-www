The Amberbio app for analyzing biological data sets. Figure of iphone with app. Available n the app store.

Data analysis using mobile devices

The Amberbio app keeps data sets and peforms calculations and visualization on the devices. This mobility it possible to look at the data at any time and place. The data can be visualized and analyzed in real time during meetings.

Data sets

The data sets must consist of samples and molecules, and a measurment for each mlecule on each sample. Missing values are allowed. Many types of data fit this description. Examples of data sets are mRNA gene expression, protein abundances, microRNA abundance. Meaurement techniques include microarrays, sequencing, qPCR, mass spectrometry, and 2D gels.

Even data sets of a non-biological origin can be used.

Simple user interface

The user interface is mostly touch based and is very easy to use. No training is needed. Each page in the app has an info button which provides some additional information if anything might be unclear.

Price

The app is a paid app. The only payment takes place at time of download. There are no adds in the app. The price is variable and can be checked at the App Store.

Privacy

There is complete privacy. No data is stored or sent to a server. All analysis is local. No login is required.

Import and export

New data sets can be imported from tab, or comma, separated files. Files can be imported through email, or through cloud storage providers such as Icloud, Dropbox or Box. Result files can be exported through email or cloud storage providers.

Projects can be exported to a file that can be used for backup and transfer to other devices.

Features

The app can perform t-tests, Mann-Whitney tests, ANOVA tests, paired t-tests, linear regression, various visualizations, 2D and 3D principal component analysis, clustering, logarithm transformation, normalization, factor elimination, factor association tables, produce missing value plots among others.

Figures can be customized, and colors can be chosen. Figures can be exported to pdf or png. Tables can be exported to txt files.

A description of the features of the app can be found on Features.
