Programming

Proper software is crucial in the research process. It is needed both to manage and keep track of samples and data in the laboratory and to answer research questions. Often, existing software packages only partially solve the particular problems in a given laboratory. We offer to write customized software solutions that increase productivity and speed in the research process. An example is computer scripts that translate one type of gene ids to another, or that annotate genes. For this example, manual annotation is possible but would take much longer time. We can also assist in developing and implementing new algorithms for research purposes.

We are particularly well versed in the programming languages Perl and Python, the database query language SQL and the statistics language R, but we offer services in other languages as well.
